Trabalho TGA2 - ANTLR4
Nomes: Cássio Deon e Roger Calderini

O trabalho foi desenvolvido no ambiente windows.
Os exercícios estão nas referidas pastas.

\EXERC_1
  Exercício 1 - Sonda
  Executar os seguintes comandos:
    antlr4 exercicio1_sonda.g4
    javac *.java

  Após isso o exercício está pronto para ser rodado.
  
  Executar o comando: java CalcDistanceSonda comandos.txt
	ou então: java CalcDistanceSonda [ENTER] [DIGITAR OS COMANDOS E INFORMAR FIM DE INSTRUÇÃO ^D OU ^Z (No windows)]

  A gramática está no arquivo exercicio1_sonda.g4
  O Listener está no arquivo CalcDistanceSondaListener.java
  E o código para execução no arquivo CalcDistanceSonda.java

\EXERC_2
  Exercício 2 - Analisador de código Java
  Executar os seguintes comandos:
    antlr4 Java.g4
    javac *.java

  Após isso o exercício está pronto para ser rodado.
  
  Executar o comando: java AnalysisJavaTool Demo.java
  
  A gramática está no arquivo Java.g4
  O Listener está no arquivo AnalysisJavaListener.java
  E o código para execução no arquivo AnalysisJavaTool.java
